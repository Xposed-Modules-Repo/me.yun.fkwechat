## 功能介绍

### 实用工具

| 功能 | 描述 |
|------|------|
| 自动领取红包 | 实时监听并自动领取红包，支持延迟设置和群聊屏蔽 |
| 自动领取转账 | 实时监听并自动收取转账，支持延迟设置和群聊屏蔽 |
| 消息防撤回 | 阻止他人撤回消息，查看被撤回内容 |

| 艾特清除换行 | 艾特时自动清除换行符号 |

## 安装说明

### 环境要求

- **Xposed 框架**: LSPosed / EdXposed
- **微信版本**: 支持微信 8.0.x 及以上版本
- **Android 版本**: Android 9.0 及以上

### 安装步骤

1. 确保已安装 Xposed 框架 (推荐使用 LSPosed)
2. 下载最新版本的 APK 文件
3. 在 LSPosed 管理器中激活 FkWeChat 模块
4. 选择微信作为作用域
5. 重启微信即可生效

## 常见问题

**Q: 模块未生效怎么办？**
> 请确保已在 LSPosed 中正确激活模块，并勾选了微信作为作用域，然后重启微信。

**Q: 某些功能无法使用？**
> 部分功能可能与微信版本相关，请确保微信已更新至最新版本。
